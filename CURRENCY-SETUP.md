# إعداد العملة - الدينار العراقي

## العملة المستخدمة

النظام يستخدم **الدينار العراقي (د.ع)** كعملة أساسية.

## الرمز والاسم

- **الرمز**: `د.ع`
- **الاسم**: `دينار عراقي`
- **الكود**: `IQD`

## التنسيق

### أمثلة على التنسيق:

```
1,000 د.ع
10,000 د.ع
100,000 د.ع
1,000,000 د.ع
```

### استخدام التنسيق:

```javascript
import { formatCurrency } from '@/lib/currency'

// مثال
formatCurrency(1000000) // "١٬٠٠٠٬٠٠٠ د.ع"
```

## الأماكن التي تستخدم التنسيق

1. **صفحة الرواتب** (`app/payroll/page.js`)
   - الراتب الأساسي
   - الخصومات
   - المكافآت
   - صافي الراتب

2. **صفحة الموظفين** (`app/employees/page.js`)
   - الراتب الأساسي للموظف

3. **جميع صفحات النظام**
   - أي مبلغ مالي يتم عرضه بتنسيق الدينار العراقي

## التحقق من التنسيق

- ✅ الرمز: `د.ع`
- ✅ التنسيق: أرقام عربية مع فواصل
- ✅ التنسيق العراقي: `ar-IQ`
- ✅ بدون كسور عشرية (أرقام صحيحة فقط)
